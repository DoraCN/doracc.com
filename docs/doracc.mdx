# Dora 中文社区

Dora 中文社区面向正在学习、使用和贡献 Dora 的中文开发者。这里会持续整理中文教程、实践案例、社区活动和交流入口，帮助新手更快找到学习路径，也帮助已有经验的开发者沉淀实践。

## 你可以在这里做什么

### 学习 Dora

如果你刚开始接触 Dora，建议先从[新手学习路线](/guide/start/learning-path)开始。它会告诉你应该先看什么、先跑什么、遇到问题去哪里查。

### 参与交流

遇到安装、数据流配置、节点开发、示例复现等问题，可以通过社区渠道交流。提问时建议附上系统环境、Dora 版本、运行命令、dataflow 配置和完整报错。

### 共建内容

欢迎贡献中文教程、示例复现、踩坑记录、文章翻译和视频笔记。哪怕只是一个很小的 demo，只要能帮助后来者少走弯路，就值得被整理进社区资料。

## 适合贡献的内容

- 新手安装和环境配置记录
- 一个最小 dataflow 的解释
- Python / Rust / C++ 节点开发笔记
- 摄像头、视觉模型、语音、机械臂等应用示例
- 从 ROS 或传统脚本迁移到 Dora 的实践经验
- 文档错别字、链接失效、命令更新等维护工作

## 社区准则

- 对事不对人，尊重不同经验背景的开发者
- 提问时尽量提供可复现信息
- 分享内容时注明来源，尊重原作者
- 不发布广告、刷屏、攻击性或违法内容

Dora 中文社区还在成长中。欢迎从一个问题、一次修正文档、一个小示例开始参与。
