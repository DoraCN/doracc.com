# 跟朵拉学机器人：从零到具身智能

你好呀！欢迎来到《跟朵拉学机器人：从零到具身智能》。

这门课会带你从**零基础**出发，亲手教会一个机器人去**看见世界、听懂人话、动手做事**——哪怕你现在还不会编程，也完全没问题。

:::info 小多说
嗨，我是**小多**！我是你在这门课里要一起打造的机器人。刚开始我什么都不会，但在你的帮助下，我会一步步长出"神经""眼睛""耳朵""大脑"和"手"。我们一起加油吧！
:::

## 先分清两个名字：朵拉 和 小多

课程名字叫"跟**朵拉**学机器人"，可主角机器人却叫**小多**，是不是有点绕？其实很简单：

- **朵拉（DORA）** 是我们要学的**技术**——一个专门为机器人和 AI 设计的"数据流"框架。它是我们造机器人的"工具与方法"。
- **小多** 是你用朵拉**亲手造出来的机器人**。它的名字就取自"朵拉（DORA）"——朵拉是那套大本领，小多是你用这套本领养大的小家伙。

一句话：**用朵拉，养小多。**

## 什么是 DORA？

DORA 的全称是 **Dataflow-Oriented Robotic Architecture**（面向数据流的机器人架构），它是一种**数据流中间件（Dataflow Middleware）**。这个词听起来吓人，但你可以先记住一个画面：

> 它就像给机器人配了一块**大黑板**——机器人身上的各个部件（眼睛、大脑、手……）都围着这块黑板，谁看到了什么就写上去，谁需要就抬头看一眼。大家靠这块黑板高效地配合。

这块"黑板"正是 DORA 让机器人变得又快又聪明的秘密。后面每一章，我们都会用"黑板与课堂"这个比喻，把复杂的原理讲成你一听就懂的大白话。

如果你听过 **ROS/ROS2**，可以先把 DORA 理解成一个**更快、更简单**的新选择——第 1 章我们会用"黑板 vs 传纸条"细讲两者的差别。

## 课程特点

- 🌱 **零基础友好**：全中文正文与代码注释，配 Python 极简速成，不会编程也能学。
- 🧠 **一看就懂**：用"黑板与课堂"的统一比喻讲原理，把抽象概念说成大白话。
- 📦 **零污染环境**：配套 **朵拉魔盒（DoraBox）** 一键准备隔离环境，内置国内镜像，三平台一致，卸载即净。
- 🔧 **真实且前沿**：基于 **DORA 1.0 RC 源码编译**，锁定版本、人人结果可复现。
- 💻 **不依赖显卡**：仿真用 PyBullet，AI 用轻量模型，**普通电脑 CPU 就能跑通**。
- 🎯 **项目驱动**：**12 章 + 8 个可运行小项目**，跟着机器人小多从收发消息，一路到看见、听说、思考、抓取。

## 你将学到什么

跟着这门课走完，你将能够：

- 理解**数据流**是怎么让机器人各部件协同工作的；
- 用 **Python** 写出自己的第一个机器人程序节点；
- 让机器人接上**摄像头看见物体**、用**麦克风听懂说话**；
- 用**仿真**让机械臂动起来，不需要买任何硬件；
- 最终完成**毕业项目**：对小多说一句话，它就能看到并抓取你指定的东西——这就是所谓的**具身智能（Embodied AI）**。
