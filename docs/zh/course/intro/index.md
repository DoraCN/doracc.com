# 从零开始学 DORA

欢迎来到《从零开始学 DORA》。

这门课会带你从**基础命令行操作**出发，亲手搭建环境、编写节点、运行数据流，最终让小莫（一个示例机器人）**看见世界、听懂人话、动手做事**。

:::info 关于命名
本课程所学的开源项目全称为 **DORA-rs**（`github.com/dora-rs/dora`），为方便阅读，课程正文统一简写为 **DORA**，两者指同一技术。
:::

## 什么是 DORA？

DORA 的全称是 **Dataflow-Oriented Robotic Architecture**（面向数据流的机器人架构），它是一种**数据流中间件（Dataflow Middleware）**。一句话概括：

> 它就像给机器人配了一块**大黑板**——机器人身上的各个部件（眼睛、大脑、手……）都围着这块黑板，谁看到了什么就写上去，谁需要就抬头看一眼。大家靠这块黑板高效地配合。

如果你听过 **ROS/ROS2**，可以先把 DORA 理解成一个**更快、更简单**的新选择——第 1 章我们会用"黑板 vs 传纸条"细讲两者的差别。

## 课程特点

- 🔧 **面向开发者**：假设你会 Linux 基础命令、能搭建开发环境，直奔主题。
- 🧠 **一看就懂**：用"黑板与课堂"的统一比喻讲原理，把抽象概念说成大白话。
- 💻 **标准命令行搭建**：Rust 工具链 + uv/Python + 源码编译 DORA，三平台一致。
- 💻 **不依赖显卡**：仿真用 PyBullet，AI 用轻量模型，**普通电脑 CPU 就能跑通**。
- 🎯 **项目驱动**：**12 章 + 8 个可运行小项目**，循序渐进，从收发消息到看见、听说、思考、抓取。

## 你将学到什么

- 理解**数据流**是怎么让机器人各部件协同工作的；
- 用 **Python** 写出自己的第一个机器人程序节点；
- 让机器人接上**摄像头看见物体**、用**麦克风听懂说话**；
- 用 **Rust** 为关键节点加速；
- 用**仿真**让机械臂动起来，不需要买任何硬件；
- 最终完成**毕业项目**：说一句话，小莫就能看到并抓取你指定的东西——这就是所谓的**具身智能（Embodied AI）**。
