# 术语中英对照表

学习中遇到不熟悉的词，来这里查。全课程统一"**中文（English）**"的写法，第一次出现时括注英文，之后用中文。

## DORA 核心概念

| 英文 | 中文 | 一句话说明 |
|------|------|-----------|
| Dataflow | 数据流 | 多个节点按输入输出连成的"值日表"，用 YAML 描述 |
| Node | 节点 | 数据流里干一件事的"同学"（独立程序） |
| Operator | 算子 | 更轻量的处理单元，住在同一运行时里 |
| Input / Output | 输入 / 输出 | 节点"看黑板"与"写黑板"的端口 |
| Topic | 主题 | 一个发、多个收的广播式通信 |
| Service | 服务 | 一问一答的通信 |
| Action | 动作 | 长任务，边做边反馈 |
| Streaming | 流式 | 连续数据，可中途打断 |
| Coordinator | 协调器 | 管理全体的"班主任" |
| Daemon | 守护进程 | 管一台机器的"组长" |
| Node Hub | 节点仓库 | 现成节点的公共库，一行即可引入 |
| Record & Replay | 录制回放 | 把数据流录成 `.drec` 文件，可离线重放调试 |

## 数据与性能

| 英文 | 中文 | 一句话说明 |
|------|------|-----------|
| Apache Arrow | Arrow | 统一的数据"书写规范"，多语言零翻译 |
| Shared Memory | 共享内存 | 大家都看得见的那块"黑板" |
| Zero-copy | 零拷贝 | 不抄写，直接对着黑板用，所以快 |
| Zenoh | Zenoh | 跨机器传数据的零拷贝"数据面" |
| Serialization | 序列化 | 把数据"打包"（传纸条要折起来的那一步） |

## 工具与环境

| 英文 | 中文 | 一句话说明 |
|------|------|-----------|
| DoraBox | 朵拉魔盒 | 桌面隔离工具，独立项目，本课程不再使用 |
| Middleware | 中间件 | 让各零件顺畅协作的"公共制度" |
| Toolchain | 工具链 | 编译运行所需的整套工具（如 Rust/Python） |
| Mirror | 镜像源 | 国内加速下载源，让安装飞快 |
| CLI | 命令行工具 | 在终端里敲命令来操作的程序（如 `dora`） |

## 机器人与 AI

| 英文 | 中文 | 一句话说明 |
|------|------|-----------|
| Embodied AI | 具身智能 | 让 AI 拥有身体，去感知并作用于真实世界 |
| Simulation | 仿真 | 用软件（PyBullet）模拟机器人和环境 |
| VLM | 视觉语言模型 | 能同时看懂图像和语言的 AI 模型 |
| Object Detection | 物体检测 | 认出画面里有什么、在哪里（如 YOLO） |
| ASR | 语音识别 | 把说的话变成文字（听） |
| TTS | 语音合成 | 把文字变成说出来的声音（说） |
| ROS / ROS2 | —（专有名） | 最有名的老牌机器人中间件，本课的"参照物" |
