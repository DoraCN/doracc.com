# 第五章 · 数据的语言：Arrow

> 本章主题：🧮 处理数据

第四章你学会了写节点、传数据，但传的都是简单的开关和数字。这一章，我们把"数据"这件事讲透。

## 本章你会学到

- **[5.1 为什么是 Arrow](./why-arrow)**：搞懂 DORA 为什么统一用 Arrow，以及零拷贝为何快。
- **[5.2 常见数据类型](./data-types)**：手把手学会造和读数字、数组、图像。
- **[5.3 多输入合并](./multi-input)**：一个节点收多路数据时，如何区分与合并。
- **[🎯 小项目②：数据处理流水线](./project-pipeline)**：搭一条"生成 → 变换 → 可视化"的完整数据流。

## 学完能做什么

- 理解 Arrow 格式与零拷贝的原理；
- 在节点间自如地传递数字、数组、图像；
- 处理多个输入，用"最新值缓存"合并它们；
- 独立搭建并运行一条数据处理流水线。

## 前置要求

- 完成[第四章](../python-node/)，能写并跑通 Python 节点；
- 会一点点 Python，NumPy 会在本章用到（环境已备好，无需手动安装）。
