# 1.5 认识小多与朵拉

到这里，第一章的概念就学得差不多了。最后我们把本课程的两位"主角"正式介绍一下，别再把它们弄混。

## 朵拉是技术，小多是机器人

- **朵拉（DORA）**：我们要学的**技术**——那套"给机器人配一块共享黑板"的数据流框架。它是**工具与方法**。
- **小多**：你用朵拉**亲手打造的机器人**，是本课程的主角。它的名字取自"朵拉（DORA）"的谐音——**朵拉是大本领，小多是你用这套本领养大的小家伙**。

一句话：**用朵拉，养小多。**

:::info 小多说
所以呀，"朵拉"是我赖以生存的本事，"小多"才是我本人～ 我们是一对好搭档！
:::

## 小多的成长之旅

接下来的每一章，我们都会给小多解锁一项新能力。它会像这样一步步长大：

> ⚡ 神经（能收发）→ ✍️ 会表达（写节点）→ 🧮 会处理数据 → 💬 学会沟通 → 👁 眼睛 → 👂🗣 耳朵与嘴 → 💪 变强壮 → 🦾 手 → 🧠 大脑（思考并行动）

到最后的**毕业项目**，你只要对小多说一句话，它就能**看到并抓取**你指定的东西——那就是它的"成年礼"。

## 下一步：先给小多准备一个"工坊"

要动手造小多，得先有一套干净、趁手的开发环境。为此我们准备了配套工具 **朵拉魔盒（DoraBox）**：打开它，就能一键备好隔离、零污染的环境，还内置了国内镜像，下载飞快。

从下一章开始，我们就用朵拉魔盒来搭建环境、编译 DORA。

## 小结

- **朵拉 = 技术（DORA 框架）**，**小多 = 你造的机器人主角**，名字同源但指代不同。
- 小多会随课程逐章解锁能力，直到完成具身智能毕业项目。
- 下一章：用**朵拉魔盒**准备开发环境，正式开工。
