# 第七章 · 视觉与感知

> 本章主题：👁 视觉

从本章开始，小莫要"睁开眼睛"了。我们搭建一条完整的视觉流水线：摄像头采集画面 → 通过 DORA 数据流传递 → YOLO 物体检测 → 结果可视化。

## 本章内容

- **[7.1 摄像头节点](./camera-node)**：用 OpenCV 读取摄像头画面，通过 DORA 节点发送
- **[7.2 图像在数据流中传递](./image-in-dataflow)**：零拷贝原理、元数据传尺寸、多路订阅
- **[7.3 Python 节点接入 YOLO](./yolo-detection)**：Python 节点直接调用 YOLOv8，输出检测结果
- **[🎯 小项目④：实时物体检测](./project-detection)**：搭建完整的视觉流水线

## 学完能做什么

- 写出摄像头节点，实时采集画面并通过数据流发送
- 用 YOLO 做实时物体检测，消费边界框数据
- 在图像上绘制检测结果，搭建多节点视觉数据流

## 前置要求

- 完成[第四章](../python-node/)、[第五章](../data/)，能写 Python 节点、处理 Arrow 数据
- 有内置或外接摄像头
- 基本的 OpenCV 概念（环境会安装好）

:::tip
本章需要摄像头硬件。如果没有物理摄像头，可以使用 USB 摄像头、手机摄像头（通过 DroidCam 等软件虚拟成 PC 摄像头）或使用图片文件替代。
:::
