# DORA 中文社区

**DORA 中文社区** 是面向中文开发者的 DORA 学习、交流与共建家园。DORA 是一个为 AI 时代打造的高性能机器人开发框架，正吸引越来越多中文开发者。我们希望把最好的中文教程、最热情的交流氛围和最开放的共建机制带给每一个想做机器人的人——无论你是学生、爱好者，还是资深工程师。

> 我们的目标：让"用 DORA 做机器人"这件事，对中文世界的每个人都**触手可及**。

## 我们提供什么

### 📚 系统的中文课程

- [**从零开始学 DORA**](/course/intro/)：面向开发者，循序渐进完成 8 个项目，直到完成具身智能毕业项目。
- [**DORA 基础教程**](/guide/start/introduction)：系统、参考式地通读 DORA 的核心概念与命令。

### 🌐 官方资料

- **DORA 官网（含中文文档）**：[dora-rs.ai](https://dora-rs.ai/) —— 入门与查阅权威资料的首选。
- **源码仓库**：[atomgit.com/dora-rs/dora](https://atomgit.com/dora-rs/dora)

### 💬 交流与答疑

- **GitHub（Issues / Discussions）**：[github.com/DoraCN](https://github.com/DoraCN) —— 提问、报 Bug、参与讨论、贡献代码。
- 许多中文开发者已是项目的活跃贡献者，中文社区在项目发展中有着实实在在的影响力。

## 发起人与 LoopFun 开源社区

DORA 中文社区由 **李扬** 发起（2025 年，北京）。他同时也是 **LoopFun 开源社区** 的发起人。

**LoopFun** 是一个面向全社会的**具身智能开源社区**，致力于**降低具身智能机器人的开发门槛**，让学生、爱好者、工程师都能轻松上手——并在动手创造的过程中**收获快乐**。

本站与课程，正是 **DORA 中文社区** 与 **LoopFun 开源社区** 携手共建的成果。

## 社区特点与展望

- **技术驱动**：成员以开发者和研究者为主，聚焦真实的实现、性能与落地。
- **生态共建**：人人既是使用者，也是贡献者与布道者，一起把 DORA 推向中文世界。
- **新兴且活跃**：项目年轻但成长飞快；随着具身智能（Embodied AI）浪潮，社区正快速壮大。

如果你对**机器人、AI 与高性能系统**充满热情，现在正是加入、共同塑造它未来的好时机。欢迎你，也欢迎一起把小莫养大！ 🤖
