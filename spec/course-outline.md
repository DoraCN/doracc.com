# 课程大纲（Course Outline）

> 12 章 + 8 个小项目，围绕虚拟机器人小莫的"能力成长"主线逐步组装，毕业项目体现具身智能。
> 对零基础学员采用"小步慢走"：每章单一焦点。
> 大纲可在写作中微调，但整体主线与项目设置以此为准。

## 能力成长线（主线）

小莫从"什么都不会"逐步长出各项能力：
⚡ 神经（收发）→ ✍️ 表达（写节点）→ 🧮 处理数据 → 💬 学会沟通 → 👁 眼睛 → 👂🗣 耳朵与嘴 → 💪 变强壮 → 🦾 手/移动 → 🧠 大脑（思考并行动）。

---

## 第一章：认识 DORA 与具身智能（概念 + 动机）
- 1.1 什么是数据流中间件？为什么具身智能需要它
- 1.2 为什么是 DORA——和 ROS2 比一比（黑板 vs 传纸条）
- 1.3 核心概念图解：节点 / 数据流 / 算子 / 共享内存 / Arrow
- 1.4 四种通信模式（仅点名，详解见第六章）
- 1.5 认识小莫与朵拉：谁是技术、谁是机器人
- 🎯 建立心智模型，无需写代码
- 写作方案见 `spec/module1-plan.md`

## 第二章：朵拉魔盒与环境搭建
- 2.1 认识朵拉魔盒：为什么需要隔离环境（零污染）
- 2.2 安装魔盒并选择工作目录
- 2.3 一键准备环境：Rust + Python 隔离与国内镜像
- 2.4 从源码编译 dora 1.0 RC（锁定 commit）
- 2.5 内置终端与环境自检
- 依赖朵拉魔盒，待工具就绪后写

## 第三章：第一个数据流——dora 小飞机（对标 ROS 小乌龟）
- 3.1 获取并编译小飞机工程（aviation）
- 3.2 `dora run` 跑起来：控制器 + 小飞机两个窗口
- 3.3 读懂 `dataflow.yml`：controller → aviation
- 3.4 看懂在流动的数据：`cmd` 与 `pose`（初识 Arrow 数组）
- 3.5 啊哈实验：删掉那行连接会怎样
- 🎯 **小项目①**：运行并玩转 dora 小飞机（**只运行体验，不写代码**）
- 小莫解锁：⚡ 神经——能收发消息
- 详细方案见 `spec/first-dataflow-aviation.md`

## 第四章：Python 节点开发基础——写你自己的第一个节点
- 4.1 节点生命周期：`Node()`、事件循环、`send_output`
- 4.2 用 Python 写一个回显/控制器节点
- 4.3 把你的 Python 节点接进数据流
- 4.4 定时器与参数配置
- 小莫解锁：✍️ 会"表达"（亲手写节点）

## 第五章：数据的语言——Arrow 与数据类型
- 5.1 为什么是 Arrow：统一的"黑板书写规范"
- 5.2 常见数据类型：数字、数组、图像
- 5.3 多输入合并
- 🎯 **小项目②**：数据处理流水线（生成 → 变换 → 可视化）
- 小莫解锁：🧮 会处理各种数据

## 第六章：节点如何对话——四种通信模式
- 6.1 Topic：发布订阅（一喊全听见）
- 6.2 Service：请求应答（一问一答）
- 6.3 Action：长任务（边做边报）
- 6.4 Streaming：流式（连续可打断）
- 🎯 **小项目③**：给小飞机加一个"问答 / 任务"交互
- 小莫解锁：💬 掌握 4 种沟通方式

## 第七章：让小莫看见——视觉与感知
- 7.1 摄像头节点（OpenCV）
- 7.2 图像在数据流中的传递（零拷贝要点）
- 7.3 用 Node Hub 接入 YOLO 物体检测
- 🎯 **小项目④**：实时摄像头物体检测 + 可视化
- 小莫解锁：👁 眼睛

## 第八章：让小莫听见与说话——语音与流式
- 8.1 麦克风与音频流
- 8.2 语音识别 ASR：**SenseVoiceSmall**（ModelScope，经 sherpa-onnx 跑 ONNX，纯 CPU）
- 8.3 语音合成 TTS（sherpa-onnx 中文 VITS / piper，待锁定）
- 8.4 用 Streaming 模式做"可打断"对话
- 🎯 **小项目⑤**：和小莫语音互动
- 小莫解锁：👂🗣 耳朵与嘴
- 全程轻量 CPU 模型，不依赖 GPU

## 第九章：加速小莫——Rust 节点与混合编程
- 9.1 为什么关键节点用 Rust；何时该用
- 9.2 用 Rust 写一个 DORA 节点（对照 Python 版）
- 9.3 Python 与 Rust 节点混合在同一数据流
- 🎯 **小项目⑥**：高频数据处理节点（Rust）+ Python 可视化
- 教学深度：**Rust 最小够用**——只讲"看得懂 + 照着改一个节点"所需，不系统教语法
- 小莫解锁：💪 变强壮（高性能）

## 第十章：让小莫动起来——控制与仿真
- 10.1 控制信号的数据流建模
- 10.2 接入 PyBullet 仿真（**SO-101 机械臂**，加载其 URDF）
- 10.3（可选硬件）真实 SO-101 机械臂 / 舵机接入方式说明
- 🎯 **小项目⑦**：键盘 / 指令控制仿真机械臂
- 小莫解锁：🦾 手 / 移动

## 第十一章：让小莫思考并行动——VLM 具身智能（毕业章）
- 11.1 大模型 / VLM 如何融入数据流（善用 Streaming）
- 11.2 "语言指令 → 感知 → 决策 → 动作"闭环设计
- 🎯 **小项目⑧（毕业项目）**：说一句话，让仿真小莫（SO-101）看到并抓取指定物体
- 小莫解锁：🧠✨ 大脑（合体）

## 第十二章：调试、部署与进阶
- 12.1 日志、可视化、常见错误排查
